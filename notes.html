📌 Five Laws of Library Science

  • ✓ Books are for use
  • ✓ Every reader his/her book
  • ✓ Every book its reader
  • ✓ Save the time of the reader
  • ✓ Library is a growing organism

📌 Classification Schemes

  • ✓ DDC (1876) - Melville Dewey
  • ✓ UDC (1895) - Otlet & La Fontaine
  • ✓ CC (1933) - Ranganathan
  • ✓ LC Classification
  • ✓ Bibliographic Classification

📌 Cataloguing Codes

  • ✓ AACR1 (1967) → AACR2 (1978)
  • ✓ AACR2 Revised (1988, 1998)
  • ✓ RDA (2010) - Resource Description & Access
  • ✓ MARC 21 Format
  • ✓ CCF - Common Communication Format

📌 Bibliometric Laws

  • ✓ Lotka's Law (Scientific productivity)
  • ✓ Bradford's Law (Scattering)
  • ✓ Zipf's Law (Word frequency)
  • ✓ Impact Factor = Citations/Articles
  • ✓ h-index, g-index, i10-index
